# GDScript Toolkit (Somni Fork)

Fork of [Scony/godot-gdscript-toolkit](https://github.com/Scony/godot-gdscript-toolkit) with minor customizations.

## Custom Modifications

- **Single blank lines**: Uses `--single-blank-lines` as the default formatting style (single blank lines between top-level statements instead of double).

## Installation

```
pip3 install git+https://github.com/SomniGameStudios/godot-gdscript-toolkit.git
```

For full documentation, usage, and reference, see the [original repository](https://github.com/Scony/godot-gdscript-toolkit).
